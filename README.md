# ♟️ 2D Chess

A sleek and fun 2D Chess game built with **React Native** and **Expo**. Play solo or with a friend, enjoy smooth visuals, sounds, and a clean interface — right from your Android device.


---

## 📱 Features

- ✅ 1 Player vs AI mode (random move logic)
- ✅ 2 Player mode (local)
- ✅ Visual highlights for valid moves and last move
- ✅ Undo and Restart options
- ✅ Game over modal with win/draw messages
- ✅ Background and custom chess piece images
- ✅ Sound effects for move, check, and win (toggleable)

---

## 🛠 Built With

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [chess.js](https://github.com/jhlywa/chess.js)
- [expo-audio](https://docs.expo.dev/versions/latest/sdk/audio/)
- Custom assets and splash screen
